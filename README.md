## ⚙️ 기능 확장 기획
### 1. 전투 시스템
- **HP 시스템**: 플레이어/적 체력 추가, HP바 표시  
- **공격 강화**: 단발 → 다중탄, 강화탄 / 공격속도 조정  
- **피격 판정**: 무적시간 및 시각효과 적용  

### 2. 적 AI 및 패턴
- **스폰 패턴**: 일렬, 좌우대칭, 랜덤 워프 등  
- **이동 AI**: 곡선, 지그재그, 추적형 등  
- **공격 패턴**: 확산탄, 회전탄, 유도탄  

### 3. 아이템 및 강화 시스템
- 공격속도 증가 / 다중탄 / 체력회복 / 보호막 / 점수아이템  
- 적 파괴 시 드랍, 일정시간 후 소멸  
- 버프 지속시간 관리  

### 4. 스테이지 및 난이도
- 스테이지별 패턴, 속도, 배경음 차등 설정  
- 난이도 변수로 자동 조정  

### 5. 사운드 및 피드백
- 효과음, 배경음, 폭발 애니메이션, 화면 흔들림  

### 6. 게임 상태 관리
- `start`, `playing`, `paused`, `gameOver`  
- 일시정지, 재시작, 로컬 점수 저장  

### 7. 입력 및 플랫폼 확장
- 모바일 터치 입력 대응  
- Shift: 스킬, Ctrl: 특수탄

## 🎨 그래픽 & 비주얼 업그레이드
- **배경**: 별, 행성, 성운 그라데이션
