# 🚀 우주 여행 슈팅 게임

## 📝 업데이트 내용

### 🎨 그래픽 & 비주얼
- **배경 효과**
  - 반짝이는 별 150개 추가 (밝기 변화 애니메이션)
  - 7가지 종류의 행성 추가 (고리 포함)
  - 우주 성운 그라데이션 효과
  
- **캐릭터 디자인**
  - 기본 사각형 → 드래곤을 탄 캐릭터로 변경
  - 날개, 머리, 몸통 등 세밀한 디테일
  - 상하로 움직이는 애니메이션 효과
  - 추진 불꽃 효과

- **적 캐릭터**
  - 단순 빨간 사각형 → 3가지 타입의 개성있는 외계인
    - 타입1: 초록색 더듬이 달린 외계인
    - 타입2: 보라색 날개 달린 외계인
    - 타입3: 금색 방어막 외계인 (체력 2)
  - 체력바 표시 (타입3)

- **이펙트**
  - 총알: 기본 노란 사각형 → 빛나는 사이언 그라데이션
  - 폭발 파티클 효과 (적 격파 시)
  - 보스 격파 시 대폭발 이펙트 (충격파, 섬광 포함)

### 🎮 게임플레이
- **보스전 추가**
  - 500m마다 보스 등장
  - 보스 체력 30, 체력바 표시
  - 보스 패턴: 좌우 이동 + 2연발 공격
  - 보스 격파 카운트 기록

- **난이도 조절**
  - 거리에 따라 게임 속도 자동 증가
  - 적 배치 시스템 개선 (8줄 → 연속 생성)

- **게임 오버 & 재시작**
  - 게임 오버 화면 추가
  - 이동 거리, 보스 격파 수 표시
  - 스페이스바로 재시작 기능

### 📊 UI 개선
- 상단 정보창 추가
  - 이동 거리 (Distance)
  - 현재 속도 (Speed)
  - 보스 격파 수 (Bosses)
- 하단 조작법 안내
- 네온 스타일의 캔버스 테두리

### 🎯 게임 시스템
- **적 생성 시스템**
  - 30프레임마다 랜덤 → 줄 단위 생성
  - 8개씩 일렬로 배치
  - 100픽셀 간격으로 연속 생성

- **충돌 감지**
  - 플레이어-적 충돌
  - 플레이어-적 총알 충돌
  - 총알-적 충돌
  - 총알-보스 충돌

- **성능 최적화**
  - 화면 밖 객체 자동 제거
  - 파티클 수명 관리
  - 효율적인 렌더링
